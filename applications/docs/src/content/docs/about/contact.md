title: Contact
description: How to reach the Sideline team.
---

The fastest ways to reach us:

- **Report a bug** — use the **Report a bug** link in the app's user menu, or go to [/about/report-a-bug/](/about/report-a-bug/).
- **General questions** — open a [GitHub discussion](https://github.com/maxa-ondrej/sideline/discussions) on the repository.
- **Security issues** — do NOT open a public issue. Email the maintainer directly (see the SECURITY.md file in the repo for the current address).

We're a small team — expect a response within a few days for most things. Security reports get prioritised.
